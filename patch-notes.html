업데이트 기록

Pixby 패치노트

새 버전이 배포될 때 다운로드 파일과 함께 이 페이지도 최신 상태로 갱신됩니다.

2026-06-11

v0.1.9

  • 환경설정에 현재 연도 기반 저작권 문구와 The Mercenary 외부 링크가 있는 `픽스비 앱 정보` 영역을 추가했습니다.
  • 기존 알림 ON/OFF를 `픽스비 상태 알림`과 `새 업데이트 알림`으로 분리했습니다.
  • 새 업데이트가 있으면 알림으로 안내하고, 설치는 앱 업데이트 화면에서 사용자가 직접 시작하도록 조정했습니다.

2026-06-11

v0.1.8

  • 업데이트 다운로드와 설치 작업을 메인 프로세스에서 계속 관리하도록 변경했습니다.
  • 트레이 팝오버가 닫히거나 업데이트 페이지를 벗어나도 업데이트가 계속 진행됩니다.
  • 업데이트 진행 상태와 취소 버튼이 있는 별도 윈도우 창을 추가했습니다.
  • 앱 실행 시 새 버전이 있으면 일반 트레이 UI 시작 전에 자동으로 업데이트를 진행합니다.

2026-06-11

v0.1.7

  • 트레이 팝오버 오른쪽 하단에 현재 연도 기반 저작권 문구와 The Mercenary 외부 링크를 추가했습니다.
  • 외부 링크는 Electron 내부 창이 아니라 기본 브라우저 새 창으로 열리도록 처리했습니다.
  • 앱 실행 중 새 버전이 발견되면 macOS 로컬 알림으로 안내하도록 백그라운드 업데이트 확인을 추가했습니다.

2026-06-11

v0.1.6

  • 더보기 메뉴의 업데이트 확인 항목을 새 앱 업데이트 페이지로 이동하도록 변경했습니다.
  • 앱 업데이트 페이지에서 현재 앱 버전을 readonly로 표시하고, 진입 시 최초 1회 자동으로 새 버전을 확인합니다.
  • 새 버전이 있으면 앱 안에서 다운로드와 설치 후 재실행을 진행하는 버튼을 표시합니다.

2026-06-11

v0.1.5

  • Developer ID signing과 Apple notarization을 통과한 정식 macOS 공개 빌드로 전환했습니다.
  • 업데이트 피드와 다운로드 링크를 v0.1.5 GitHub Release asset으로 갱신했습니다.
  • 공증 없는 임시 ad-hoc 빌드 안내 문구를 제거했습니다.

2026-06-09

v0.1.4

  • 트레이 팝오버의 픽스비 방 배경을 창밖 도심이 보이는 방 내부 분위기로 개선했습니다.
  • OS 시간에 따라 새벽, 낮, 해질녘, 밤 배경이 바뀌도록 했고 도심 건물 구도는 동일하게 유지했습니다.
  • 상태 메시지, 돌봄 피드백, 알림 문구가 사용자가 설정한 픽스비 이름을 사용하도록 수정했습니다.
  • 상태 메시지 박스가 픽스비 애니메이션을 가리지 않도록 아래로 내리고 상태 게이지 하단 여백을 늘렸습니다.
  • 공증은 아직 적용하지 않은 임시 ad-hoc signed 빌드라 첫 실행 시 macOS 보안 허용이 필요할 수 있습니다.

2026-06-08

v0.1.3

  • v0.1.2 공개 파일의 깨진 서명 문제를 피하기 위해 ad-hoc signed 빌드로 다시 배포했습니다.
  • 공증은 아직 적용하지 않은 임시 빌드라 첫 실행 시 macOS 보안 허용이 필요할 수 있습니다.

2026-06-08

v0.1.2

  • 메뉴바 픽스비의 표시 높이를 26px로 조정해 상하 여백을 약 1px 수준으로 줄였습니다.
  • 공개 다운로드 후 다른 Apple Silicon Mac에서 손상됨 경고가 확인되어 다운로드를 일시 중지했습니다.
  • 다음 공개 빌드는 Developer ID signing과 notarization 검증 후 다시 배포합니다.

2026-06-08

v0.1.1

  • 픽스비 앱 아이콘과 공개 페이지 로고를 새 `logo-pixby.png` 기준으로 교체했습니다.
  • 메뉴바 애니메이션을 앱 창의 애니메이션 규칙과 같은 스케줄러 기반으로 맞췄습니다.
  • 낮, 저녁, 밤 시간대별 애니메이션 비중을 조정했습니다.
  • 빈 스프라이트 프레임을 제거해 앱 창에서 픽스비가 깜빡이는 현상을 줄였습니다.
  • 오버레이 창을 필요할 때만 생성하도록 바꿔 기본 대기 RAM 사용량을 낮췄습니다.

2026-06-07

v0.1.0

  • macOS 메뉴바 픽셀 반려묘 앱의 첫 로컬 MVP를 구성했습니다.
  • 첫 실행 온보딩, 고양이 타입 선택, 이름 설정, 상태 저장을 추가했습니다.
  • 먹이 주기, 쓰다듬기, 청소, 재우기 등 기본 돌봄 행동을 구현했습니다.
  • 메뉴바 트레이와 작은 오버레이 창을 통해 상태 확인과 설정을 할 수 있게 했습니다.
